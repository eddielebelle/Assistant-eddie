# Eddie Assets

This directory contains visual assets for the Eddie project.

## Files

- `architecture.png` - System architecture diagram
- `mqtt-flow.png` - MQTT message flow sequence diagram
- `eddie-banner.png` - Project banner/header image (optional)

## Generating Diagrams

The Mermaid diagram source files are in the root directory:
- `architecture.mermaid`
- `mqtt-flow.mermaid`

You can render these using:
- [Mermaid Live Editor](https://mermaid.live/)
- GitHub (automatically renders in README)
- VS Code with Mermaid extension
